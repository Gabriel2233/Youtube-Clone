##  Installation :octocat:
Make sure your have Node.js installed on your computer.

Clone the repo:

git clone https://github.com/Gabriel2233/React-Chat-App.git

After entering inside the project directory, start the project:

```bash
yarn
```

```bash
yarn start 
```

That's it!
